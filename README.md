# 成德國小古詩背誦及英檢學習任務網

這是一個給家長與國小學生使用的線上學習網站，內容包含古詩背誦、英檢 600 單字、英檢句型練習、學習進度紀錄與星光學院闖關冒險。

## 網站連結

[開啟學習網站](https://andersonlin0107v2.github.io/chengde-learning/)

## 使用手冊

[開啟 PDF 使用手冊](https://andersonlin0107v2.github.io/chengde-learning/readme.pdf)

## 主要功能

- 古詩分級背誦與朗讀
- 英檢 600 單字練習
- 英檢句型練習
- 大字模式與專注模式
- 使用者學習紀錄
- 總熟記率與分級熟記率
- 星光學院闖關冒險
- 阿不寬小狗學習小幫手
- 每日抽籤與回到最上頁功能

## 使用注意事項

學習紀錄主要保存在使用者目前的裝置與瀏覽器中，不會自動上傳到 GitHub。

若更換手機、清除瀏覽器資料、使用無痕模式或改用不同瀏覽器，原本紀錄可能不會出現。建議家長與學生固定使用同一台裝置與同一個瀏覽器。
